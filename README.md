# 🌌 Aether-Core { Cyber-Luxury Portfolio }

Welcome to **Aether-Core**, an immersive, high-performance portfolio hub designed for the modern architect of code. This project balances precise digital aesthetics with ultra-smooth 60fps animations, creating a "Cyber-Luxury" experience.

![Aether-Core Preview](https://github.com/user-attachments/assets/ae-preview-placeholder.png)

## ✨ Core Features

- **Cyber-Luxury Motion System**: Powered by Framer Motion, featuring high-frequency text scrambling, holographic card tilts, and orchestrated exit/entry transitions.
- **Glassmorphism UI**: A custom-built design system using Tailwind CSS v4, optimized for depth, transparency, and high-contrast digital interfaces.
- **Next.js 15 + React 10 Architecture**: Leveraging React 19 Server Components and the new App Router for maximum performance.
- **Responsive HUD Navigation**: Interactive sidebar and navbar designed to feel like a futuristic operating system (FUI).

## 🚀 Technical Stack

- **Framework**: [Next.js 15.1+](https://nextjs.org/) (App Router)
- **Engine**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Motion**: [Framer Motion 12](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/) & [Material Symbols](https://fonts.google.com/icons)

## 🛠️ Getting Started

### 1. Initialize Nodes
```bash
npm install
```

### 2. Enter Development Mode
```bash
npm run dev
```

### 3. Build Production Core
```bash
npm run build
```

## 📂 Project Structure

- `src/app/` - Primary routing and page definitions (Terminal, Network, Transmissions).
- `src/components/ui/` - High-performance motion primitives and "AetherAura" backgrounds.
- `src/components/layout/` - System architecture including the HUD Sidebar and Navbar.
- `src/components/home/` - The Aether Portal (Hero) and visual node components.
